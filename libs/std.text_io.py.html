std/text_io.py

As of 1.4.8 classes will no longer be supported in the language level, we may provide a python extension to use classes, or you may need to implement it yourself.

Due to this, _im_repr methods of functions will no longer be used to format objects and may result in the object, being printed raw. 

-- Prints the values separated by spaces
   and also prints a newline. --
io:println(variadic:value)

-- Similar to io:println but with no
   newline. --
io:print(variadic:value)

-- Takes in input and sets a variable --
io:input(name, text?)

-- Takes in a list of text, if a number
   is encountered treat it as an ascii code. --
io:rawoutput(variadic:value)
